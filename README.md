
EnergieFitness-34 — Gym Website



A modern, responsive gym \& fitness website built with React + TypeScript using Vite.

The project focuses on clean UI/UX, fast performance, and a real-world business layout suitable for a personal fitness brand.



Live Demo: https://energie-fitness.vercel.app/



Overview



This project presents a production-style frontend for a gym business, showcasing programs, pricing, transformations, and contact information. It is designed as a Single Page Application (SPA) with clear navigation and responsive layouts.



Key Features



Single Page Application (SPA)



Responsive design (mobile \& desktop)



Clean, fitness-themed UI



Structured pages:



Home



Programs



Pricing



Transformations



Free Trial



Contact



Fast builds and hot reload with Vite



Deployed on Vercel



Tech Stack



Frontend: React, TypeScript



Build Tool: Vite



Styling: CSS



Linting: ESLint



Deployment: Vercel

Getting Started (Local Setup)



Clone the repository



Install dependencies:



npm install





Start the development server:



npm run dev





Open:



http://localhost:5173



Deployment



The application is deployed on Vercel.

Redeployments can be triggered via GitHub integration or manual deploys from the Vercel dashboard.
